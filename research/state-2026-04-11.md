### 1. Original 7-stage directive vs alt 4-phase directive

Two different 9-hour plans are now on the table. They produce meaningfully different demos.

| Dimension | Original (7-stage) | Alt (4-phase loop) |
|---|---|---|
| Scope | All 7 stages: Plan→Design→Build→Test→Review→Document→Deploy | Just 4: Plan→Build→Test→Review |
| Codex demo mode | Replay only (asciinema) | Live, inside Blaxel self-healing loop |
| Sacred moment | Blaxel egress kill (security framing) | Blaxel self-healing loop iteration (engineering framing) |
| Redis use | Agent Memory Server + Context Retriever (two products) | Redis 8 Vector Sets as semantic cache + distributed locks |
| Sponsor coverage | All 5 sponsors anchored, full table | Tighter, focused on the loop's 4 stages |
| Risk | Lower — replays + mocks | Higher — live Codex inside live Blaxel |
| Demo gold | Memory drift catch + Blaxel kill (2 moments) | Self-healing loop visibly iterating 3x (1 hero moment) |

**Three Es analysis:**

| Option | Elegant | Efficient | Effective |
|---|---|---|---|
| **Original** | 7 columns is busy but coherent | 9h budget is realistic; replay mode de-risks Codex | All 5 sponsors visibly used; covers safety + memory + GTM angles |
| **Alt** | Tighter (4 columns); cleaner narrative | Riskier — live Codex eats time if it stalls | Stronger engineering judge appeal; weaker breadth of sponsor coverage |
| **Hybrid** ⭐ | 7 stages visible in UI, but 4 are deeply built; rest are stubs | Build the 4 stages well, stub the other 3 — saves H6/H7/H8 budget | Best of both — full sponsor table + the self-healing loop demo |

**Recommended:** Hybrid. Build the 4 critical stages deeply (Plan/Wordware, Build/Codex, Test/Blaxel, Review/Redis adversarial). Stub Design, Document, Deploy as visual columns that just animate without real logic. Use Codex live inside the Blaxel loop (the alt's hero moment) AND keep the Blaxel egress kill as a SECOND demo beat (the original's hero moment) by triggering the kill on a failed test that tries to phone home. **You get both demo gold moments in one beat.**
